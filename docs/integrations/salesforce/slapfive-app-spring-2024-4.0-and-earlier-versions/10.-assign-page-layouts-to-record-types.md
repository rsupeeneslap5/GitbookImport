description: >-
  This step ensures that the Page Layouts for the SlapFive Customer Influence
  and SlapFive Customer Influence Activity custom objects are mapped to the
  right Record Types.
---

# 10. Assign Page Layouts to Record Types

### SlapFive Customer Influence object

Go to **Setup** > **Objects and Fields** > **Object Manager** and click on the **SlapFive Customer Influence** object.

Go to **Record Types** and click the **Page Layout Assignment** button in the upper right.

Click the **Edit Assignment** button.

![Assign Page Layouts to SlapFive Customer Influence object Record Types](<../../../.gitbook/assets/Screenshot 2022-04-03 214922.png>)

* [ ] Select all the cells in the **Customer Content Share** column, and then select **SlapFive Share Layout** from the **Page Layout to Use** dropdown box.
* [ ] Select all the cells in the **Customer Reference Request** column, and then select **SlapFive Request Layout** from the **Page Layout to Use** dropdown box.
* [ ] Select all the cells in the **Customer Referral** column, and then select **SlapFive Referral Layout** from the **Page Layout to Use** dropdown box.
* [ ] Click **Save**.

### SlapFive Customer Influence Activity object

Go back to **Object Manager** and edit the **SlapFive Customer Influence Activity** object.

Go to **Record Types** and click the **Page Layout Assignment** button in the upper right.

Click the **Edit Assignment** button.

![Assign Page Layouts to SlapFive Customer Influence Activity object Record Types](<../../../.gitbook/assets/Screenshot 2022-04-03 215614.png>)

* [ ] Select all the cells in the **Customer Content Shared & Viewed** column, and then select **SlapFive Share Activity Layout** from the **Page Layout to Use** dropdown box.
* [ ] Select all the cells in the **Customer Reference Activity Performed** column, and then select **SlapFive Request Activity Layout** from the **Page Layout to Use** dropdown box.
* [ ] Select all the cells in the **Customer Referral Activity** column, and then select **SlapFive Referral Activity Layout** from the **Page Layout to Use** dropdown box.
* [ ] Click **Save**.
