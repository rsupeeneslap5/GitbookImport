# 1. Install the Package

{% hint style="info" %}
Note: If you already have a SlapFive Salesforce App package version 3.1 or earlier installed, you need to uninstall that package before installing this package. If you currently have version 3.4 or later installed, installing this package will upgrade you to the latest version.
{% endhint %}

Go to this URL to bring up the **Install SlapFive App** screen. If not already logged in, you will be prompted to log into the Salesforce org into which you want to install the SlapFive Salesforce App:

{% embed url="https://login.salesforce.com/packaging/installPackage.apexp?p0=04t8b000000qWxc" %}

Select the **Install for All Users** option, check the acknowledgement, and click **Install**.
