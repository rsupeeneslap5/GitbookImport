# 5. Enable My Domain

Go to **Setup** > **Company Settings** > **My Domain**.

Follow the steps in this document to enable My Domain: [https://help.salesforce.com/articleView?id=domain\_name\_setup.htm\&type=5](https://help.salesforce.com/articleView?id=domain_name_setup.htm\&type=5).\
\
For more information, see [https://developer.salesforce.com/docs/atlas.en-us.lightning.meta/lightning/intro\_reqs\_my\_domain.htm](https://developer.salesforce.com/docs/atlas.en-us.lightning.meta/lightning/intro_reqs_my_domain.htm).
