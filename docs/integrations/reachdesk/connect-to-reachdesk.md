# Connect to Reachdesk

To protect the security and privacy of your Reachdesk data, SlapFive's Embedded Integration eliminates the need for you to email or message us with your authentication credentials. You or your Reachdesk Admin can simply connect right from within SlapFive. You only have to do this once and it will be used for all Reachdesk automations. The login data is encrypted and stored within the Embedded Integration.

1\. In SlapFive, go to the Admin menu, click **Settings**, and click on the **Integrations** sub-tab.

2\. Click the **Reachdesk Connection** box to expand it. Enter the following fields and click **Connect**.

* [ ] **Reachdesk API Token:** Paste your Reachdesk API Token. The API Token can be created and revoked by an Organization Admin in the Reachdesk platform at [https://app.reachdesk.com/api\_tokens](https://app.reachdesk.com/api_tokens).

Once you are successfully connected, the button should now say **Disconnect**, and you should see a green <mark style="color:green;">**Connection success**</mark> message.
