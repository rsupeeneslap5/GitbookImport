# Creating File or Link type Boards

**First Go to Boards**

1\. Add new and select the template either [Customer Case Study Post with Link or File Story : Anonymous Customer ](https://globalization-partners.slapfive.com/client/templates/edit/clcglovcb000035aheyf0d7ku)or Identified if you know the customer name.

2\. Put in your headline - matching case study title

3\. then go to the tab stories shown below and drag and drop your story you want&#x20;

4\. then go to Custom fields and put Yes to show in Salesforce
